To check which versions are maintained to get security fixes.
If you need security updates, please update to one of the supported version.

| Version | Supported          |
| ------- | ------------------ |
| 8.x   | :white_check_mark: |
| <= 7.x   | :x:                |

after they have been analysed and a fix is available.
